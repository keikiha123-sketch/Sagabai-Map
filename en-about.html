About This Website

Kure Runners Map is a map site for runners in Kure City, Hiroshima Prefecture. Kure City has a mountainous and coastal landscape, and there are many scenic routes and easy-to-run courses, but this information is not well organized. This site displays running courses that you can actually run on a map in an easy-to-understand way, and allows you to check the start point, route, distance, and features at a glance. We aim to provide an easy reference when you're wondering "where should I run today?" Also, we have designed the site to be simple and easy to use for beginners and experienced runners alike. We plan to gradually add more courses so that this site is useful for both tourists visiting Kure and local runners who run regularly. We hope that Kure Runners Map will be a starting point for making your running more enjoyable and easier to continue.
